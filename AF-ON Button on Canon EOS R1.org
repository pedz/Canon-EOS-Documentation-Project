#+TITLE:  AF-ON Button on Canon EOS R1
#+AUTHOR: Perry Smith
#+EMAIL:  pedz@easesoftware.com

The [[file:svg/icon_Button_AF_ON_w60.svg]] (AF-ON) button on the R1
has a half press and a full press setting.  In still shooting mode,
the button itself can be set to one of seven different settings
referred to in the manual as [[file:svg/icon_AF_ON_1.svg]] (AF-ON 1) -
[[file:svg/icon_AF_ON_7.svg]] (AF-ON 7).  In Movie shooting mode in
one of four settings referred to as [[file:svg/icon_AF_ON_1.svg]]
(AF-ON 1) - [[file:svg/icon_AF_ON_4.svg]] (AF-ON 4)

For Still Shooting, the seven possible settings are:

 * AF-ON 1 :: Half-press has selections and defaults to
   [[file:svg/icon_metering-af-start.svg]] (Metering and AF start);
   Full-press is disabled

 * AF-ON 2 :: Half-press is disabled; Full-press has selections and
   defaults to [[file:svg/icon_metering-af-start.svg]] (Metering and AF
   start)

 * AF-ON 3 :: Half-press set to
   [[file:svg/icon_metering-af-start.svg]] (Metering and AF start);
   Full-press has selections and defaults to Switch ES cont. shooting
   modes

 * AF-ON 4 :: Half-press set to
   [[file:svg/icon_metering-af-start.svg]] (Metering and AF start);
   Full-press has selections and defaults to Eye Detection AF

 * AF-ON 5 :: Half-press set to
   [[file:svg/icon_metering-af-start.svg]] (Metering and AF start);
   Full-press has selections and defaults to Start/stop whole area AF
   tracking

 * AF-ON 6 :: Half-press set to
   [[file:svg/icon_metering-af-start.svg]] (Metering and AF start);
   Full-press has selections and defaults to Off

 * AF-ON 7 :: Half-press set to
   [[file:svg/icon_metering-af-start.svg]] (Metering and AF start);
   Full-press has selections and defaults to Off

   AF start)

   [[file:svg/icon_metering-af-start.svg]] (Metering and AF start);
   Full-press has selections and defaults to Off
