#+TITLE:  Canon EOS Documentation Project
#+AUTHOR: Perry Smith
#+EMAIL:  pedz@easesoftware.com

Initially this project addressed two pain points for me and I think
others.

  1. The functions that can be assigned to buttons are not documented.
     I wanted a place to document them.

  2. I can't find the menu that I'm looking for.  I wanted to create a
     flat list of the camera's menu system into a text file that could
     be loaded onto a phone and searched.

At this point, both of those objectives have been accomplished for the
Canon EOS R1 and the Canon EOS R5 Mark II.

The project uses [[https://orgmode.org][Org Mode]] for the source and
a small Elisp function in publish.el to publish the index, HTML pages,
and UTF-8 files.

I hope to add more cameras and I hope that others will participate but
so far, I have had some comments that this is a great idea but no one
has actually helped.
